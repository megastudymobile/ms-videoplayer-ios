# API 레퍼런스

### 핵심 클래스

| 클래스 | 설명 |
| --- | --- |
| [KollusPlayerView](/dev-guide/kollus-mobile-app/sdk/ios/api-reference/player-view/) | 콘텐츠 재생, 화면 출력 제어, 북마크 관리, 이벤트 델리게이트 설정 등 플레이어의 모든 기능을 제어하는 재생 엔진 |

| [KollusStorage](/dev-guide/kollus-mobile-app/sdk/ios/api-reference/kollus-storage/) | 오프라인 재생을 위한 콘텐츠 다운로드·삭제·조회를 관리하는 클래스 |
